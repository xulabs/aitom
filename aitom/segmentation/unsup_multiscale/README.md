# Unsupervised Multi-Scale Segmentation of Cellular Cryo-Electron Tomograms

This repository contains the code associated with our CVPR 2026 Spotlight paper **"Unsupervised Multi-Scale Segmentation of 3D Subcellular World with Stable Diffusion Foundation Model"**.
---

## Quick Start

### 1. Install dependencies

```bash
pip install -r requirements.txt
```

### 2. Place your `.mrc` files

Put your tomogram files inside:

```
input/raw_data/
```

### 3. Register your tomograms

Edit `tomogram_config_testing.py` to describe each tomogram:

```python
tomogram_configs = {
    'TS_0001': {                      # Tomogram Name
        'slab_range_start': 200,      # First slice to process
        'slab_range_end':   300,      # Last slice to process
        'tomogram_size':  (928, 960)  # (height, width) in pixels
    },
}
```

### 4. Run the pipeline

```bash
python unified_tomogram_pipeline.py --input_path input/raw_data --output_path output/results
```

For a quick debug run with fewer steps:

```bash
python unified_tomogram_pipeline.py --train_steps 10
```

### Output Structure

The pipeline produces two types of ouputs— 1) binary 2D masks for the membranes or membrane-like structures for the selected slabs and 2) detected macromolecule coordinates in (x,y,z) format within the selected slab range. After the pipeline finishes, results are saved under:

```
output/
└── <TOMOGRAM_NAME>/
    ├── membrane_masks/               # Membrane segmentation labels as 2D masks for the selected slabs
    └── macromolecule_coordinates/    # Detected macromolecule coordinates (x, y, z) within the selected slabs
```

 These labels can be used to train supervised models for faster segmentation and macromolecule localization.

---

#### Membrane Segmentation with Supervised UNet using our prediction


The membrane masks generated by our method for the selected slabs in the training tomogram can be used as training labels for the supervised refinement model inside `membrane_seg/`, and the trained model is then run for inference to produce the membrane segmentations for other slabs in the tomogram or other tomograms. See [membrane_seg/README.md](membrane_seg/README.md) for full instructions.

---

#### Macromolecule Localization with DeepETPicker using our prediction

The macromolecule coordinates generated by our method for the selected slab ranges in the training tomogram can be used as training labels for supervised macromolecule localization methods, such as, **[DeepETPicker](https://github.com/cbmi-group/DeepETPicker)**. DeepETPicker an interactive GUI tool for particle picking in cryo-ET tomograms. Set it up following its repository instructions, load your `.mrc` files, use the coordinates generated by our method as `ground-truths` in `.csv` file. Once trained using the mrc files and ground-truths, the trained DeepETPicker model (which is a lighweight UNet based model) is used for inferring the macromolecule locations in the entire training tomogram as well as the other tomograms in the dataset.

## Directory Overview

| Path | Description |
|---|---|
| `unified_tomogram_pipeline.py` | Main entry point — runs the full unsupervised pipeline |
| `tomogram_config_testing.py` | Register tomograms and their slice/size parameters |
| `membrane_seg/` | Supervised membrane segmentation (trains on pseudo labels) |
| `input/raw_data/` | Place `.mrc` tomogram files here |
| `output/` | All results written here |
| `code_visulaize_macromolecule_coords.py` | Visualize detected macromolecule coordinates |
